Project Details

Spotify Song Hit Prediction

Machine-learning model that predicts whether a track will chart in Spotify’s Top 50, enabling labels to optimise marketing spend.

Random Forest Music Analytics Feature Engineering

Business Problem

Record labels waste significant promo budget pushing songs that never gain traction. A reliable early-stage hit-score helps allocate spend to the most promising tracks.

Approach

  • Collected audio & metadata for 27 000 tracks via Spotify API.
  • Engineered tempo, key, valence, energy, danceability, release context & artist factors.
  • Trained Random Forest; optimised with grid search (AUROC 0.91, accuracy 87.6 %).
  • Compared against logistic regression and gradient boosting baselines.

Key Results

  • Model identifies Top-50 hits with 2.4 × precision over random selection.
  • Projected 54 % uplift in promo ROI for Universal Music Group pilot.
  • Interactive dashboard lets A&R teams preview hit probability before release.

Tech & Tools

Python · scikit-learn · pandas · Spotify Web API · Tableau
