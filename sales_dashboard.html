Tableau Project  |  Sales Performance​ Dashboards

Introduction

This project delivers two interactive Tableau dashboards—one for Sales Performance and one for Customer Insights—so sales managers and executives can monitor KPIs, spot trends, and drill into product-level performance without writing a single query.

Dashboard Purpose

The Sales dashboard provides an at-a-glance view of year-over-year sales, profit, and quantity. The goal is to highlight monthly and weekly fluctuations so stakeholders can quickly answer: “Where are we outperforming? Where do we need to intervene?”

Key Requirements & How I Solved Them

  • KPI Overview
    • Built a high-level tile for each KPI (Sales, Profit, Quantity) that compares current YTD vs. prior YTD and uses colour-coded arrows to flag positive or negative change.
  • Sales Trends
    • Added an interactive line chart with dual-year overlay.
    • Included dynamic “best / worst month” call-outs so spikes are visible the moment the data refreshes.
  • Product Sub-Category Comparison
    • Used a butterfly bar to show Sales vs. Profit side-by-side for every sub-category.
    • Embedded tooltips reveal profit margin %, turning the viz into a quick diagnostic of low-margin products.
  • Weekly Sales & Profit
    • Constructed a heat-map calendar that benchmarks each week against the current-year average—weeks above the line pop in indigo, below-average weeks in muted grey.
    • Added a parameter toggle so users can switch between absolute $ values and % deviation.
