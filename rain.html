Project Details

Mumbai Temperature & Rain-Prediction

Built next-day forecasts for temperature (robust regression, R² 0.84) and rain probability (neural-net, 90 % accuracy) using eight years of daily Mumbai weather data.

Neural Network Robust Regression Time-Series Weather Data Lag Features

Business Problem

Mumbai’s volatile monsoon disrupts transport, retail demand and energy usage. Accurate next-day weather predictions help businesses optimise staffing, logistics and advertising spend.

Approach

  • Eight-year daily dataset: temp, humidity, pressure, wind, rain flag.
  • Created one- & two-day feature lags; encoded rain Yes/No → 1/0.
  • Temperature → robust OLS (R² 0.84); Rain → 7-feature neural net (2 × hidden layers).

Key Results

  • Regression explains 83.8 % of temperature variance.
  • Rain model: 90 % accuracy, 87 % sensitivity, 92 % specificity.
  • Interactive dashboard provides 24 h forecast to partners.

Tech & Tools

Python · pandas · statsmodels · Keras · scikit-learn · matplotlib
